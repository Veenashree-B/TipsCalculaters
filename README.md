# Tip Calculator App 💸

A sleek and functional Android application designed to quickly calculate tips and total bill amounts. This project demonstrates proficiency in handling user input, managing dynamic application state, and building interactive user interfaces using modern Android development practices.

## ✨ Features
* **Real-time Calculation:** Instantly calculates the tip amount and total bill as the user enters data, providing immediate visual feedback.
* **Interactive Inputs:** Utilizes Material Design 3 components, including custom `TextField` inputs for numerical data entry and a `Switch` toggle for a "round up" feature.
* **Advanced State Management:** Showcases a strong understanding of Compose state handling (`remember`, `mutableStateOf`), state hoisting, and unidirectional data flow to ensure UI consistency.
* **Optimized User Experience:** Implements smart keyboard handling (e.g., showing the number pad for bill input and integrating the "Next" or "Done" action keys) for a seamless mobile experience.

## 🛠️ Tech Stack
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Design System:** Material Design 3
* **Build System:** Gradle
